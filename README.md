# Cache

A lightweight, professional clipboard caching app template for macOS.

## Features
- Menubar-first UI
- Fast, local-only storage
- Simple hotkey workflow

## Quick Start
Open `Cache.xcodeproj` in Xcode and run the `Cache` scheme.
